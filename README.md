## Build Setup
### setup .env
```bash
cp .env.dist .env
```
Now add your flare key to `.env`

### install dependencies
```bash
npm install
```

### serve with hot reload at localhost:3000
```bash
npm run dev
```

### build for production & upload sourcemap 
```bash
npm run build

# launch server
npm run start
```
